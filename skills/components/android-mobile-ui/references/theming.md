# Theming

## Material 3

Use `MaterialTheme.colorScheme`, `MaterialTheme.typography`, and `MaterialTheme.shapes` in composables.

## Color

- Define brand colors in the theme, not in individual screen bodies.
- Ensure contrast in light and dark mode.
- Use color to support meaning, not as the only indicator.
- In health apps, use alert colors sparingly and consistently.

## Typography

- Use Material typography tokens.
- Do not scale font size with viewport width.
- Avoid negative letter spacing.
- Keep compact panels visually quiet.

## Premium Feel

Premium mobile UI usually comes from restraint:

- consistent spacing
- precise alignment
- clear type hierarchy
- high-quality empty/loading/error states
- predictable interaction
- minimal decorative noise
